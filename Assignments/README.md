## Coverage

These assignments comprehensively cover the following topics:

- **8086 Architecture**: Internal architecture, pin diagram, memory segmentation
- **Instruction Set**: Data transfer, arithmetic, logical, control transfer instructions
- **Addressing Modes**: Immediate, direct, indirect, indexed addressing
- **Interfacing**: Peripheral interfacing, 8255 PPI, 8259 PIC
- **Advanced Processors**: Introduction to 80286, 80386, and Pentium processors
